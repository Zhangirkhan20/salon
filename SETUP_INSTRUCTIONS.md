# 🚀 Queen Bee Salon - Инструкции по запуску

## Что это?
Полностью готовый профессиональный сайт салона красоты на React + Vite с:
- ✅ Красивым дизайном (золото/черное/белое)
- ✅ Мобильной версией
- ✅ Интерактивной системой записи
- ✅ Интеграцией с WhatsApp
- ✅ Всеми необходимыми компонентами

## Быстрый старт (3 шага)

### Шаг 1: Скачать Node.js (если нету)
Скачай с: https://nodejs.org/ (версия 16+)

Проверь в терминале:
```bash
node -v
npm -v
```

### Шаг 2: Открыть проект и установить зависимости
```bash
# Открой папку queen-bee-salon в VS Code
# Потом в терминале VS Code выполни:

npm install
```

Ждешь 1-2 минуты пока установятся все пакеты.

### Шаг 3: Запустить сайт
```bash
npm run dev
```

Откроется: `http://localhost:5173`

## Что дальше?

### Редактировать данные салона
Открой `src/salonData.js` и измени:
- Название салона
- Номер телефона (WhatsApp)
- Адрес
- Услуги
- Мастеров
- Отзывы

### Изменить дизайн
Открой `src/styles.css` и измени:
- Цвета (ищи `#d4af37` для золота)
- Шрифты
- Размеры

### Развернуть на сайт (GitHub Pages или Vercel)

#### GitHub Pages (Бесплатно)
1. Создай GitHub аккаунт
2. Создай новый репозиторий "queen-bee-salon"
3. В VS Code терминале:
```bash
git config --global user.name "Ваше имя"
git config --global user.email "your.email@gmail.com"
git init
git add .
git commit -m "Initial commit"
git remote add origin https://github.com/YOUR_USERNAME/queen-bee-salon.git
git push -u origin main
```
4. Зайди в Settings репозитория → Pages → выбери "main" → Save

Готово! Сайт будет на: `https://YOUR_USERNAME.github.io/queen-bee-salon`

#### Vercel (Еще проще)
1. Создай Vercel аккаунт (https://vercel.com)
2. Нажми "New Project"
3. Выбери свой GitHub репозиторий
4. Нажми Deploy

Готово! 🎉

## 🐛 Если что-то сломалось

1. **Сайт не открывается:**
   - Проверь что `npm run dev` запустился
   - Открой http://localhost:5173

2. **Ошибка при npm install:**
   ```bash
   # Удали node_modules и попробуй снова
   rm -rf node_modules
   npm install
   ```

3. **Ошибка в коде:**
   - Открой F12 в браузере (Developer Tools)
   - Посмотри Console tab
   - Расскажи что пишет ошибка

## 📞 WhatsApp номер

Поменять номер в `src/salonData.js`:
```javascript
export const salonData = {
  phone: '77021113346',  // ← Измени на свой
  // ...
}
```

## 📊 Услуги и мастера

Все в `src/salonData.js`:
- Добавь новую услугу в `services: []`
- Добавь нового мастера в `masters: []`
- Добавь отзыв в `reviews: []`

## 🎨 Цвета

В `src/styles.css` в самом начале:
```css
:root {
  --primary-gold: #d4af37;     /* Золото */
  --primary-black: #1a1a1a;    /* Черный */
  --primary-white: #ffffff;    /* Белый */
}
```

Измени эти коды на свои цвета.
